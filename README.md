# Phonics Sonics

This repo contains a minimal, kid-friendly one-page phonics site (JPG images + WAV/MP3 audio).
Place your images at assets/images/letters/a.jpg ... and audio at assets/audio/a.wav (or .mp3).

How to run locally:
- Serve the directory with a static server (python -m http.server or npx http-server) and open index.html.

Deploy:
- Push to GitHub and enable Pages on the main branch.
